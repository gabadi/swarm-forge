## Gotchas

- **Never push, create issues, or open PRs against `unclebob/swarm-forge` (the upstream remote).** All issue tracking, commits, and PRs go to `gabadi/swarm-forge` (origin). Use `gh issue create --repo gabadi/swarm-forge` explicitly, or set the default with `gh repo set-default gabadi/swarm-forge` at session start.
